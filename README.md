# Мой День - Приложение для здоровья

Веб-приложение для отслеживания здоровья, созданное в рамках лабораторной работы по программной инженерии.

## Функциональность

### Основные возможности:
- 📊 **Дашборд здоровья** - обзор всех показателей на главном экране
- 😴 **Отслеживание сна** - запись и история продолжительности сна
- 💧 **Контроль воды** - отслеживание потребления воды с дневной целью
- 👟 **Подсчет шагов** - запись физической активности
- 😊 **Настроение** - выбор и отслеживание эмоционального состояния
- 💊 **Лекарства** - напоминания о приеме лекарств
- 📈 **Статистика** - просмотр истории по всем категориям
- 👤 **Профиль** - настройка личных данных

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- LocalStorage для хранения данных
- Font Awesome для иконок
